BUILD A MENOPAUSE IMPACT TRACKER - SINGLE HTML FILE
====================================================

This is a prompt to build a complete, working menopause impact tracker as a SINGLE HTML FILE with embedded CSS and vanilla JavaScript. No backend. No database. No complexity. Just one file.

---

## QUICK WINS (User gets these in first 60 seconds)
✓ Sign up with email (instant)
✓ Complete first daily check-in (30 seconds)
✓ See dashboard update in real-time
✓ Track how menopause affects work, relationships, energy, mental clarity
✓ All data saved automatically to their browser (no login needed next time)

---

## TECH REQUIREMENTS
- Single HTML file (index.html)
- Embedded CSS (in <style> tag)
- Vanilla JavaScript (in <script> tag)
- Browser Local Storage for data persistence
- Mobile-first responsive design
- No external dependencies (no npm, no CDN imports)
- Professional, clean, calming design (purples, blues, soft grays)

---

## FEATURES TO BUILD

### 1. WELCOME / LOGIN PAGE (First screen)
- Simple form: Email, Password
- Button: "Sign Up" or "Log In"
- Store user email in local storage
- After login, show dashboard
- Simple logout button in settings

### 2. DAILY CHECK-IN PAGE
Form fields (in this order):

**Quick Scores (dropdowns or sliders)**
- Mood: 1-5 (Terrible, Struggling, Average, Good, Great)
- Brain Fog: 1-10 slider
- Hours of Sleep Last Night: number input (decimal, e.g., 5.5, 6, 7)
- Patience: 1-10 slider (1=Wanted to scream, 10=Very patient)
- Bleeding: dropdown (None, Spotting, Light, Medium, Heavy)
- HRT Taken Today?: Yes/No checkbox

**Impact Scores (1-10 sliders - this is the differentiator)**
- Work Impact: How much did menopause affect your work? (1-10)
- Relationship Impact: How much did menopause affect your relationships? (1-10)
- Energy Impact: How much did menopause affect your energy? (1-10)
- Mental Clarity Impact: How much did menopause affect your mental clarity? (1-10)

**Optional**
- Notes: text area (optional)

**Design:**
- Auto-fill today's date
- If user already checked in today, LOAD existing data and allow edit
- Submit button: "Save Check-In"
- Mobile-friendly (no horizontal scroll)
- Should take 30 seconds to complete

### 3. HRT TRACKER PAGE
Form to add/view HRT regimen:

**Fields:**
- Medication Name (e.g., 'Estradot', 'Oestrogel')
- Estrogen Dose (e.g., '50mcg', '1mg')
- Progesterone Name (optional, e.g., 'Utrogestan')
- Progesterone Dose (optional)
- Administration Type: dropdown (Patch, Gel, Tablet, Spray)
- Frequency (e.g., 'Daily', 'Every 3 days')
- Start Date: date picker
- Notes (optional)

**Features:**
- Show current regimen below form
- Allow editing existing regimen
- Show "Last Updated" date

### 4. DASHBOARD PAGE
Quick summary of user's data:

**Today's Status Card:**
- "Today's Check-In: [Mood] • [Brain Fog] • [Sleep X hrs]"
- If not checked in today, show "Complete today's check-in" button

**Last 7 Days Summary (simple cards with large numbers):**
- Average Sleep: X hours
- Average Brain Fog: X/10
- Average Patience: X/10
- Days with Bleeding: X
- HRT Adherence: X% (days HRT taken / 7)

**This Month:**
- Total Bleeding Days: X
- Total Check-Ins: X/[number of days in month]

**Current HRT:**
- Medication, dose, frequency

**Design:**
- Large readable numbers
- Simple cards
- Clean layout
- Mobile responsive

### 5. BLEEDING CALENDAR PAGE
Simple calendar view showing bleeding days:

**Features:**
- Show current month calendar
- Each day shows bleeding level: none (gray), spotting (light pink), light (pink), medium (dark pink), heavy (red)
- Tap a day to edit bleeding level
- Show cycle pattern (days between bleeding starts)
- Show bleeding duration (how many days per episode)

### 6. SETTINGS PAGE
- Display user email
- Change password option
- Logout button
- Delete all data option (warning: "This cannot be undone")
